SERVER-PUBLIC DASHBOARD - MODIFICATIONS SUMMARY
================================================

Modified Files:
1. app.py
2. templates/dashboard.html

NEW FEATURES IMPLEMENTED:
==========================

1. LICENSE TYPE SECTIONS
   - Added license_type field to database (discord_vm, nitro_puller)
   - Dashboard now has 3 tabs: "All Licenses", "Discord VM", "Nitro Puller"
   - When generating licenses, you can select the type from dropdown
   - Table shows license type as a badge for each key

2. NOTES FUNCTIONALITY
   - Added admin_notes field to database
   - Click any license row to view detailed information in a modal
   - Modal shows all license details including:
     * License key, type, status
     * Days, HWID
     * Created, activated, expired dates
     * Customer email and notes
     * Recent activity logs
   - Add/edit admin notes for any license
   - Notes are saved instantly and visible in the table (truncated to 20 chars)

3. EXPORT/DOWNLOAD FEATURES
   - Download Unused Keys button - exports all unused licenses to CSV
   - Download Active Keys button - exports all active licenses to CSV
   - Download All Keys button - exports all licenses to CSV
   - Downloads respect current tab filter (Discord VM or Nitro Puller)
   - CSV includes all license information

4. BULK KEY DOWNLOAD
   - When generating multiple keys (count > 1), a "Download Bulk Keys" button appears
   - Click to download all newly generated keys as a text file
   - Button appears for 10 seconds after generation
   - One key per line format for easy import

DATABASE CHANGES:
=================
- Added 'license_type' column (TEXT, default: 'discord_vm')
- Added 'admin_notes' column (TEXT)
- Auto-migration runs on first startup if columns don't exist

NEW API ENDPOINTS:
==================
- POST /admin/update_notes - Update admin notes for a license
- GET /admin/get_license_details/<license_key> - Get detailed license info
- GET /admin/export_licenses?status=<status>&type=<type> - Export licenses as CSV

UNCHANGED FEATURES:
===================
All existing functionality remains the same:
- License activation API
- License verification API
- Admin authentication
- Delete licenses
- View activation logs
- Auto-refresh dashboard
- Session management

HOW TO USE:
===========
1. Extract the RAR file
2. Install requirements: pip install -r requirements.txt
3. Run: python app.py
4. Login with admin credentials (admin/changeme123)
5. The database will auto-update with new columns

IMPORTANT NOTES:
================
- Existing licenses will default to "discord_vm" type
- You can manually update license types if needed
- Notes are optional and can be added/edited anytime
- Export files include timestamps in filenames
- All security features remain unchanged
